CONECTANDO HTML CON CSS

Hay 3 formas:

  • Forma 1: Dentro del html usar las etiquetas:

    <style> Aqui va el codigo css</style>

  • Forma 2: Crear un archivo externo y usar el

    <link rel="stylesheet" href"aqui va direccion y nombre del archivo css">

  • Forma 2: dentro de una etiqueta usar el estilo en linea

    <div style="codigo css""> </div>

SINTAXIS DE CSS <>

Se realiza de la siguiente forma:
selector {
propiedad1: valor
propiedad2: valor
propiedadN: valor
}

SELECTORES DE CSS

Selector Elementales:

  • Selector universal: *{propiedad1: valor}
  • Selector etiqueta: h1,h2,p,etc{propiedad1: valor}
  • Selector etiqueta: h1,h2,p,etc{propiedad1: valor}

Selector De Atributo:

  • id: id del elemento:
    Por ejemplo tenemos el elemento

    <div id="titulo">

    El css seria: #titulo{propiedad: valor}
  • clase: clase del elemento:
    Por ejemplo tenemos el elemento

    <div class="tarjeta">

    El css seria: .tarjeta{propiedad: valor}

COLORES

Hay 4 formas

  • nombre del color: red,blue,tomato,yellow,etc
  • Codigo hexadecimal: #11acc1,#238491,etc
  • RGB: rgba(0-255,0-255,0-255,0-1) Ese ultimo valor es la transparencia va del 0 al 1 separado por punto
  • HSL: hsl(0-100%,0-100%,0-100%)

BACKGROUND - FONDO

Existen los siguientes:

  • background-color:
    Ejemplo: Selector{background-color:red}

  • background: linear-gradient(angulo medido en deg, color - De donde arranca medido en %,color - De donde arranca medido en %, etc):
    Ejemplo: Selector{ background: linear-gradient(0deg, green 20%, yellow 70% ,red 90%)}

  • background-image: url(link):
    Ejemplo: Selector{ background-image: url(img/fubuki.jpg);}

    • Para que no se repita usamos el:
    background-repeat: no-repeat
    Podemos usar: no-repeat,repeat-x, repeat-y

    Para la pocision usamos:
    background-position: bottom right;
    "bottom, right,top,left, o podemos combinar estos"

    USAR TODO ESTO EN UNA SOLA PROPIEDAD BACKGROUND:
    background: red url(img/fubuki.jpg) no-repeat bottom right;

    EN CASO DE IMAGEN TAMAÑO DEL BACKGROUND:
    background-size: ancho alto;
    background-size: 100%;
    background-size: cover;     "Rellena todo el espacio "
    background-size: contain; "No importa el tamaño del contenedor hace que la imagen no se salga"

    Para efecto que nos siga (PARALLAX) usamos:
    background-attachment: fixed;
